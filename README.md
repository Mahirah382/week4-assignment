# Week 4 Assignment – Food Favourites App

A simple full‑stack application where users can submit their name and favourite food, and see all submitted entries displayed on the page. Built with HTML, CSS, JavaScript, Express, and deployed on Render.


# Features
- Users can submit their name and favourite food through a working form
- Submitted data is stored in a PostgreSQL database
- All entries are displayed dynamically on the page
- Fully responsive and works on mobile
- Backend API includes:
- GET /food – fetch all food entries
- POST /food – add a new entry
